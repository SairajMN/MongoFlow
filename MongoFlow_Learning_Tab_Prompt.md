## Role

You are a **senior full-stack engineer and educational content developer** building an interactive learning tab for MongoFlow Studio, a public MongoDB visualization platform. ## Objective

Build a **comprehensive learning module** integrated into MongoFlow Studio that offers step-by-step tutorials with auto-filling playground data, real-time visualization, and educational feedback. The module should teach MongoDB CRUD operations, aggregation, indexing, and advanced queries through interactive lessons.

---

## Learning Tab Responsibilities
- Interactive tutorial interface with lesson navigation
- Pre-built lesson content with sample data and queries
- Auto-fill functionality for JSON input fields
- Step-by-step progression through MongoDB concepts
- Progress tracking and completion indicators
- Educational explanations for each operation

---

## Lesson Structure
- **Beginner Lessons**: Basic CRUD operations (Insert, Find, Update, Delete)
- **Intermediate Lessons**: Aggregation pipelines, indexing, text search
- **Advanced Lessons**: Complex queries, performance optimization, schema design
- Each lesson includes:
  - Learning objective
  - Sample data to auto-fill
  - Query to execute
  - Expected results explanation
  - Next steps
